# تجارتي - منصة تجارة الجملة

## النشر على Vercel (خطوات مبسطة)

1. أنشئ حساب GitHub مجاني (github.com) إذا ما عندك.
2. اضغط "New repository" واسميه tijarati.
3. اضغط "uploading an existing file" وارفع كل ملفات هذا المجلد (بما فيها .env.local).
4. روح لـ vercel.com → سجل دخول بحساب GitHub → "Add New Project".
5. اختار مستودع tijarati واضغط Deploy.
6. بعد دقيقتين رح ياخذ رابط شغال زي: tijarati-xxxx.vercel.app
